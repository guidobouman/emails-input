Emails Input demo page

The emails input is fully stylable and does not take opinions on how to do so. Semantically, it stays as close to a regular input field as possible and uses default DOM elements to provide free accessbility. Below are a couple of example forms. This first is styled to match a specific brand: Miro. The other is unstyled.
